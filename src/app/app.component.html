Custom Pipes Example

Truncate Pipe

Description: The Truncate Pipe shortens a long string to a specified length and appends a suffix (like "..." or any other string). This is useful when displaying excerpts or limiting text length in your UI.

Original Text: "{{ longText }}"

Truncated Text (Limit: 20 characters): "{{ longText | truncate:20:'...' }}"

Multiply Pipe

Description: The Multiply Pipe multiplies a number by a specified factor. This is useful for dynamically calculating values in templates, such as adjusting quantities or scaling numbers.

Original Number: {{ number }}

Multiplied by 3: {{ number | multiply:3 }}

Append Pipe

Description: The Append Pipe adds a specified string to the end of another string. This is useful for appending additional text or labels to existing strings without modifying the original data in the component.

Original Text: "{{ title }}"

Appended Text (" - Angular Rocks!"): "{{ title | append:appendText }}"
